# Student-Study-App
Study app for CS 307


To use the template

## Step 1:
Clone the Directory
`git clone https://github.com/GavinMcCormack912/Student-Study-App.git`
## Step 2:
Get client dependencies
```
cd client
npm install
```

## Step 3:
Get server dependencies
```
cd server
npm install
```

## Step 4:
Launch the project:
`npm start`
