# Student-Study-App
Study app for CS 307


To use the template

## Step 1:
Clone the Directory
`git clone https://github.com/GavinMcCormack912/Student-Study-App.git`
## Step2:
Enter the client directory
`cd client`

## Step2:
Run the following to update dependencies:
`nmp install`

## Step 4:
Launch the project:
`npm start`
